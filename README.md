# cronometro-online
Fiz esse projetinho bem simples para poder controlar o tempo de momentos que possuem um tempo limitado. A princípio fiz para um retiro da Igreja Católica para controlar o tempo dos pregadores, entretanto pode-se ser usado para apresentações e etc. Nessa versão, devido a discrição que se deve manter ao extrapolar o tempo de uma pregação,
