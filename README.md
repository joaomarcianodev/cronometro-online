# Cronômetro Online
Fiz esse projetinho bem simples para poder controlar o tempo de momentos que possuem um tempo limitado. A princípio criei no intuito de usar em um retiro da Igreja Católica para controlar o tempo dos pregadores, entretanto pode-se ser usado para apresentações e etc.

Nessa versão, devido a discrição que se deve manter ao extrapolar o tempo de uma pregação, não será emitido nenhum som.
Além disso o app também funciona 100% offiline para você baixar e usar em locais que não possuem internet.
